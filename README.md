# 🚗 Traffic Accident Prediction

## 📌 Description
This project uses Machine Learning to predict traffic accident occurrences based on various factors such as road conditions, weather conditions, and traffic-related data.

---

## 📊 Dataset Information
The dataset contains the following features:
- Weather
- Road Type
- Time of Day
- Traffic Density
- Speed Limit
- Driver Alcohol, Age, Experience
- Road Condition

### 🎯 Target Variable
- Accident: 0 (No Accident)
- Accident: 1 (Accident)
